# Profile 1 — Basic GPS Telemetry

Basic location fix from a GPS-equipped endpoint.

## Packet Types

| Type | Name       | Direction | Description                  |
|------|------------|-----------|------------------------------|
| 1    | `location` | e->s      | Standard GPS position report |

## Type 1 — Location

`fix` is always transmitted. `lat`, `lon`, `alt`, and `accuracy` are omitted when the fix state does not support them — see the GPS Fix Convention in the booklet.

| CBOR Key | Field Name  | Type      | Optional | Description                                                    |
|----------|-------------|-----------|----------|----------------------------------------------------------------|
| 0        | `profile`   | uint8     |          | Always `1` for this profile                                    |
| 1        | `id`        | bytes (8) |          | Endpoint device identifier                                     |
| 2        | `mid`       | uint8     |          | Rolling message ID; server deduplicates on `(mid, timestamp)` |
| 3        | `ack`       | bool      |          | `true` if endpoint requests acknowledgment                     |
| 4        | `type`      | uint8     |          | Always `1` for location                                        |
| 5        | `timestamp` | uint32    |          | Unix timestamp in seconds                                      |
| 6        | `fix`       | uint8     |          | GPS fix type: `0` no fix, `1` 2D fix, `2` 3D fix             |
| 7        | `lat`       | float32   | yes      | Latitude in decimal degrees (present when fix >= 1)            |
| 8        | `lon`       | float32   | yes      | Longitude in decimal degrees (present when fix >= 1)           |
| 9        | `alt`       | float32   | yes      | Altitude in meters (present when fix = 2)                      |
| 10       | `accuracy`  | float32   | yes      | Horizontal accuracy estimate in meters                         |

### Example — 3D fix

```json
{
  "profile": 1,
  "id": "a3f1c2b4d5e6f7a8",
  "mid": 42,
  "ack": false,
  "type": 1,
  "timestamp": 1622547800,
  "fix": 2,
  "lat": 45.678,
  "lon": -123.456,
  "alt": 100.0,
  "accuracy": 5.0
}
```

### Example — no fix

```json
{
  "profile": 1,
  "id": "a3f1c2b4d5e6f7a8",
  "mid": 43,
  "ack": false,
  "type": 1,
  "timestamp": 1622547860,
  "fix": 0
}
```
